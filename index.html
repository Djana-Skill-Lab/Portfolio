Projet Power BI - Analyse Foot

Ce projet présente une analyse complète de données footballistiques avec visualisations interactives, statistiques des joueurs et performances d'équipes.

Cliquez sur le bouton ci-dessous pour télécharger le fichier .pbix :

Télécharger le projet Power BI

Contenu de l'analyse

Ce dashboard Power BI inclut :

  • Statistiques détaillées des joueurs
  • Performances des équipes
  • Visualisations des matchs et résultats
  • Analyse comparative entre ligues
  • Indicateurs de performance clés

Comment utiliser

Pour visualiser ce projet Power BI :

  1. Téléchargez le fichier .pbix en cliquant sur le bouton ci-dessus
  2. Ouvrez le fichier avec Microsoft Power BI Desktop
  3. Explorez les différentes visualisations et filtres
  4. Interagissez avec les graphiques pour découvrir les insights

Configuration GitHub Pages

Pour rendre votre portfolio accessible en ligne :

  1. Allez dans l'onglet "Settings" de votre repository
  2. Dans le menu de gauche, cliquez sur "Pages"
  3. Sous "Source", sélectionnez la branche "main"
  4. Cliquez sur "Save"
  5. Votre site sera accessible à : https://votrenomutilisateur.github.io/freelance/
