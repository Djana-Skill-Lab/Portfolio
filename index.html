Projet Power BI - Analyse Commerciale

Ce projet présente un tableau de bord complet d'analyse commerciale avec visualisations interactives, segmentation des données et indicateurs de performance clés.

Cliquez sur le bouton ci-dessous pour télécharger le fichier .pbix :

Télécharger le projet Power BI

Comment utiliser

Pour visualiser ce projet Power BI :

  1. Téléchargez le fichier .pbix en cliquant sur le bouton ci-dessus
  2. Ouvrez le fichier avec Microsoft Power BI Desktop
  3. Explorez les différentes visualisations et filtres

Modification du nom de fichier

Après avoir téléversé votre fichier Power BI, vous devez modifier le code :

  1. Remplacez "NOM-DU-FICHIER.pbix" dans le code par le nom exact de votre fichier
  2. Cliquez sur "Commit changes" pour sauvegarder
