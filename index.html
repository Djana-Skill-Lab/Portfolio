Activation de GitHub Pages GitHub Pages Activation

Pour rendre votre portfolio accessible en ligne :

  1. Allez dans l'onglet "Settings" de votre repository
  2. Dans le menu de gauche, cliquez sur "Pages"
  3. Sous "Source", sélectionnez la branche "main"
  4. Cliquez sur "Save"
  5. Attendez quelques minutes que le site se déploie
  6. Visitez https://djana-skill-lab.github.io/Portfolio/

To make your portfolio accessible online:

  1. Go to the "Settings" tab of your repository
  2. In the left menu, click on "Pages"
  3. Under "Source", select the "main" branch
  4. Click on "Save"
  5. Wait a few minutes for the site to deploy
  6. Visit https://djana-skill-lab.github.io/Portfolio/
